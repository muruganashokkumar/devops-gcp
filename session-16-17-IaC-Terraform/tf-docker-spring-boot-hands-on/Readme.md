
### How to Run

1. **Initialize Terraform**

   ```bash
   terraform init
   ```

2. **Plan**

   ```bash
   terraform plan
   ```

3. **Apply**

   ```bash
   terraform apply -auto-approve
   ```

4. **Access App**
   Visit: `http://localhost:5000`

---

### Cleanup

```bash
terraform destroy -auto-approve
```
